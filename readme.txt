# How to use
Unzip project zip.
Recommend use install virtualenv to not clutter actual system.
With virtualenv installed, run 'virtualenv venv' in folder with project, then
run 'source venv/bin/activate' on Unix. Then run 'pip install -r requirements.txt'
Note: Should be running python 3.4. Then run 'python application.py' and browse
to http://localhost:8080 in your browser.
